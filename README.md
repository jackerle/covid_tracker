# Covid Tracker by. jackerle
## Version 1.0.1
## Created with React+leafletjs

### installed
#### 1. clone this repository
``` 
$git clone https://github.com/jackerle/covid_tracker
```

#### 2. install required module (Use NPM)
```
$npm install 
```

#### 3. run with
```
$npm start
```
